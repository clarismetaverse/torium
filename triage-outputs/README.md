# Triage Outputs

This folder stores selected TORIUM triage output JSON files that are worth keeping in GitHub as evidence, examples, or benchmark runs.

Local raw outputs are generated by the triage pipeline under:

```text
outputs/triage/
```

Because local outputs may be noisy or large, only curated files should be copied here and committed.

Recommended filename format:

```text
YYYY-MM-DD-search-name-short-note.json
```

Example:

```text
2026-06-18-large-homes-fractioning-first-run.json
```

Do not commit files containing API keys, private investor data, or sensitive personal information.
